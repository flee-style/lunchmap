# lunchmap
■作成期間：11/22～11/27

■開発環境：Anaconda3、Django2.2.5、Python3.6.9

■なぜ作成したか：Djangoでテンプレートビューを用いたサイト作りを覚えたかったため。また、ユーザー管理機能も作成したかったため。
